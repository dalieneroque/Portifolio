<h2>📌 Sobre o Projeto</h2>

<p>
Este é meu portfólio pessoal desenvolvido com <strong>Blazor WebAssembly</strong>, criado com o objetivo de apresentar
minhas habilidades técnicas, projetos e trajetória na área de desenvolvimento de software.
</p>

<p>
Apesar do meu foco principal ser <strong>Desenvolvimento Backend com .NET</strong>, desenvolvi este projeto também como
forma de fortalecer minha base em <strong>HTML, CSS e estruturação de interfaces</strong>,
buscando uma visão mais completa da aplicação.
</p>
